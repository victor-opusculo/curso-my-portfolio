
# Aula 1

## Tecnologias usadas

- Next.js
- TailwindCSS
- SQLite
- Node.js
- Typescript
- DrizzleORM

## Projeto

Um site portfólio para desenvolvedor de software usando Next.js, com apoio de TailwindCSS e Typescript.

## Estrutura

Site público com:
- Página de blog;
- Página de projetos;
- Página de ferramentas (usadas em projetos);
- Núvem de tags para posts de blog e projetos;

Site privado, protegido com senha, com:
- Edição de posts de blog;
- Edição de projetos;
- Edição de ferramentas;
- Cadastro de mídias (arquivos anexos em geral);
- Página de configurações para alterar dados de contato, nome de usuário e senha.

## Testes do curso

Serão de múltipla escolha, após cada módulo, com 5 questões cada.
